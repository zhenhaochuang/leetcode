# 项目介绍

主要是leetcode热门题解和其他常用算法(排序，字符串，栈和队列等)的代码实现

### 组织结构

``` lua
com.zhenhc.leetcode
├── easy -- leetcode简单题目
├── medium -- leetcode中等题目
├── hard -- leetcode困难题目
├── sorts -- 排序算法
└── strings -- 字符串算法
```
